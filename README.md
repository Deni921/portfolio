# Portfolio

Static portfolio website (plain HTML/CSS/JS, no build step).

- **Live site:** https://deni-mekollari.vercel.app
- **Source:** the site lives in [`deploy/`](deploy/) — Vercel's project Root Directory is set to `deploy`.

## Deploying

Pushing to `main` auto-deploys to production via Vercel's GitHub integration.
Pull requests get their own preview URLs.
